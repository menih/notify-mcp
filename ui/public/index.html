How to get Google OAuth credentials
  1. Open Google Cloud Console
  2. Create a new project (or select existing)
  3. Go to APIs & Services → Library, enable Gmail API
  4. Go to OAuth consent screen → choose External, fill in app name
  5. Go to Credentials → Create → OAuth 2.0 Client ID
  6. Choose type: Web application
  7. Under Authorized redirect URIs, add exactly:
    http://localhost:3737/auth/google/callback
  8. Copy the Client ID and Client Secret below
